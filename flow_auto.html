🤖 AI秘書 & 自動化パイプライン

エージェントが仕事を回す仕組みをClaude Coworkで構築するフロー

自動化パイプラインの例

📧
reply-assist：メール・Slack返信の半自動化
Slack・Gmailの未返信メッセージを取得して、文体ルール（丁寧語・感謝先出し など）に従った返信下書きを自動生成。確認してから送るので暴走しない。
Slack/Gmail MCP で未返信取得
→
文体ルール付きで下書き生成
→
確認画面で承認
→
送信 or 修正
🔄
claude-sync / 3拠点自動同期
ローカルPC・外付けSSD・GitHub の3拠点をドライラン→承認→本実行のフローで安全に同期。SSD依存から脱却するため5リポをGitHub Private化も完了。
rsync dry-run で差分確認
→
承認
→
本実行（削除なし）
→
git pull/push で3拠点揃える
⚡
カスタムSkill（スラッシュコマンド）自作
繰り返す作業をSkillとして登録すると「/〇〇」1コマンドで全自動実行。GASプッシュ・レポート生成・同期など何でもスラッシュコマンド化できる。
作業フローをチャットで依頼
→
Skill（.mdファイル）として保存
→
次回から /skill-name で1発実行
