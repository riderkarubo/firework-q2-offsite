🌐 Chrome拡張機能の開発フロー

ブラウザ操作を自動化するChrome拡張を、コード知識ゼロでClaude Coworkと一緒に作るプロセス

作った Chrome拡張

📌
Bulk Pin Tabs
開いている複数タブを選択してワンクリックで一括ピン留め。毎朝の定番タブ管理を自動化。
manifest.json / popup.html / background.js
🤖
エージェント完了報告拡張
Claude Coworkの作業完了を検知して、デスクトップ通知+Slack通知を自動送信する拡張。
manifest.json / content.js / notify.js
📄
PDFスライド取り込み拡張
Google Slide上でPDFを開き、ページを画像として一括取り込み・整形できる拡張。
manifest.json / content.js / utils.js
