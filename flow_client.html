MCCM・ロレアル・Asics・Samsung など案件ごとのClaude Cowork活用パターン

具体的な活用ケース

MCCM — 年次報告会
📊 年次報告会の提案資料をAIと共同制作
毎年の報告会資料制作を、構成設計からHTML化まで全工程Claude Coworkと共同作業。データを貼ると分析コメントも自動生成。
①データ・方針をチャットで共有
→
②章立て・構成案をAIが提案
→
③HTML資料を自動生成
→
④品質ゲート（自己レビュー）
→
⑤GitHub Pagesで即公開
Claude Coworkへの指示例
「MCCM年次報告会資料を作りたい。視聴者KPIはXX万人、リピーター率は前年比+5pt。強みは〇〇、課題は〇〇。エグゼサマリ→KPI→施策→提案の構成でHTMLにして」
成果：資料制作時間が従来比1/3に短縮。修正も「ここをこう直して」で即反映。
ロレアル — 見積もり
💰 見積もりスプレッドシートを松竹梅GASで自動生成
単価・数量・プランを変えるだけで全計算が自動更新されるGASを構築。clasp連携でClaude Coworkから直接pushできる体制に。
①プラン構成・単価をチャットで指定
→
②GASコードを自動生成
→
③clasp pushで即反映
→
④微修正もチャットで即対応
Claude Coworkへの指示例
「ロレアル向けライブ配信見積もりGASを修正して。梅プランに追加項目「コメントモデレーション ¥50,000」を加えて、合計行と消費税行も自動更新」
成果：修正20回以上をコピペなし・clasp push のみで対応。手貼りミスゼロ。
Asics / マンチェス — 提案書
📄 ライブ配信提案書・ピッチデックをHTMLで高速制作
パワポを一から作る代わりに、HTMLで提案書を作成。OGPタグ・印刷対応・SSO保護まで品質ルール付きで自動適用。
①提案の方針・数値をチャットで共有
→
②HTML提案書を自動生成
→
③品質ゲート（OGP/印刷/SSO）
→
④GitHub Pagesで共有URL発行
Claude Coworkへの指示例
「Asics向けライブ配信提案書HTMLを作って。競合比較・ROI試算・実施スケジュールの3章構成。Fireworkテンプレートに合わせたデザインで」
成果：提案書制作が1〜2日→2〜3時間に。URLを共有するだけでクライアントがどこからでも見られる。

クライアント業務で使う主なClaude Cowork機能

📝
HTML自動生成
提案書・報告書・レポートをコードゼロで制作
⚡
GAS自動生成
スプレッドシート・スライドの定型作業を自動化
📋
品質ルール自動適用
表記統一・OGP・印刷対応を毎回チェック
🔌
GitHub Pages連携
完成したらURLを発行してクライアントに即共有
