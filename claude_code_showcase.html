何をやっているか

🏢
クライアント業務の効率化
MCCM / ヤマダデンキ / ロレアル / Starbucks ほか
  • 年次報告会の提案資料をAIと共同制作（MCCM）
  • スキルトレーニング進捗表をGASで自動生成（ヤマダデンキ）
  • Pythonでスプレッドシートテンプレートを自動セットアップ
  • 提案書・レポートのHTML化・スライド化をコード生成で量産
  • ChatFAQツール・AIエージェント報告書などクライアント向けツール開発
⚡
Google Apps Script 自動化
Sheets / Slides / Gmail の繰り返し作業をゼロに
  • 請求書の自動作成・自動抽出（PDF→スプレッドシート）
  • アンケートをGoogleフォームへ自動変換
  • Google SlideへPDFを一括取り込み
  • 質問回答まとめをレイアウト付きでSlides出力
  • Shortcutキー一覧をスライドに自動整形
🛠️
自作ツール・アプリ開発
Claude Codeがコードを書き、自分は仕様を決める
  • ライブ配信QAピックアップツール（視聴者質問を自動抽出）
  • LS分析レポート自動化アプリ
  • スケジュール・タイムキープアプリ
  • コメント分析レポート自動生成
  • HTMLToolsでUIコンポーネントを量産
🤖
AI秘書 & 自動化パイプライン
エージェントが仕事を回す仕組みづくり
  • AI秘書：返信・タスク管理・情報整理を自動化
  • reply-assist：メールやチャットの返信を半自動化
  • claude-sync-workspace：ワークスペースを自動同期
  • カスタムSkills（スラッシュコマンド）を自作して繰り返し作業を1コマンドに
  • MCPサーバーでClaude Codeの能力を拡張（Studio One連携など）
🌐
Chrome拡張機能の開発
ブラウザ上の作業をワンクリックで
  • Bulk Pin Tabs：複数タブを一括ピン留め
  • AIエージェント作業完了報告拡張
  • Google Slide上でPDF取り込み拡張
  • ブラウザ上でClaude Codeが直接操作・自動化
📚
ナレッジ化・ドキュメント整備
やり方を資産として残す
  • Claude Code活用指南書を社内ドキュメントとして整備
  • 各プロジェクトにCLAUDE.md（AIへの指示書）を配置
  • セッション引き継ぎ・チェックポイントで文脈を跨いで継続
  • 過去の作業パターンをSkill化して再利用

Claude Codeならではのポイント

⚡
コードを書かずにコードを作る
GAS・Python・HTML・Chrome拡張を、自分でコードを書かずに仕様だけ伝えて生成
🔁
繰り返し作業をゼロにする
請求書・レポート・スライド生成など毎月の定型業務をスクリプト化して自動実行
🤖
複数AIが並列で動く
ultrawork / ralph モードで複数エージェントが同時に異なるタスクを実行
📋
CLAUDE.md で AI に文脈を記憶させる
プロジェクトごとのルール・背景をAIに「永続記憶」させて毎回説明ゼロに
🛡️
AIがAIをレビューする
生成エージェント → 評価エージェント のループで品質を自動担保
🔌
MCP でどこでも繋がる
Google Drive / Notion / Slack / Studio One など外部ツールをClaude Codeに接続
💡
「AIに仕事を任せる」ではなく「AIと一緒に仕事する」
Claude Codeは単なる「コード生成ツール」ではなく、
計画・実装・レビュー・修正・納品まで一緒に走るAIパートナーです。
繰り返しの仕事を仕組みに変えて、価値ある仕事に集中できる環境をつくっています。
